
# Hướng dẫn cài đặt





## Installation


##Cài đặt ứng dụng bên phía frontend
```bash
  cd .\frontend
  npm install
  Đặt các biến môi trường trong file .env theo mẫu file env.developer
```

Chạy ứng dụng
```bash
  cd .\frontend
  npm run build
```

##Cài đặt ứng dụng bên phía backend
```bash
  cd .\backend
  npm install
  Đặt các biến môi trường trong file .env theo mẫu file env.developer
```

Chạy ứng dụng
```bash
  cd .\backend
  npm run start
```
